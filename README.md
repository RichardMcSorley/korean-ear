# Korean Ear

Cordova, React, Material-UI, Redux, React-Router, Jest

## Install

`npm install`

`npm install -g cordova`

## Run

**Run in Local Browser**

`yarn start`

**Run on Android Device**

`yarn run:android`

**Run Tests**

`yarn test` or watch tests with `yarn test -- --watch`

**Build**

`yarn build:prod` or `yarn build:dev`
